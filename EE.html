Asset Performance Management (APM)

Asset Performance Management (APM) – это технология, которая позволяет управлять эффективностью актива на основе оценки и прогноза технического состояния, рисков и последствий отказов, с помощью оптимизации планов воздействий на оборудование исходя из ключевых показателей эффективности (KPI).

Мониторинг потребления энергии:

Учет данных по потреблению энергии (автоматически по счетчикам расхода), по косвенным показателям (распределение обобщенных данных по потреблению по нескольким объектам по заданной логике), вручную); расчет удельных показателей расхода энергии в зависимости от производственного процесса, режимов и условий эксплуатации энергопотребляющего оборудования; построение Энергетической Базовой Линии

База наилучших технологий:

Проработанные решения в области энергоэффективности с описанием затрат, эффектов, окупаемости

Кадровые ресурсы по энергоэффективности:

Организационная структура, резюме, программы обучения и пр.
